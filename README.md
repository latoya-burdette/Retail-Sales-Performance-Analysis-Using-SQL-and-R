# Retail Sales Performance Analysis Using SQL and R

This project analyzes retail sales data using SQL and R.  
The dataset was imported from Excel, stored in a SQLite database, queried using SQL, and visualized with ggplot2 to explore revenue trends across customers, products, categories, and regions.
